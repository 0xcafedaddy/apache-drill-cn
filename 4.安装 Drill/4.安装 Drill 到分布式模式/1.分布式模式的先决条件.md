# 1.分布式模式的先决条件

你可以安装 Apache Drill 到一个或更多的节点到集群环境中。

## 准备

在你安装 Drill 到集群之前，确保你的集群准备了如下环境：
  * [Oracle JDK version 7](http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html)（必需）
  * 正在运行的 [ZooKeeper 集群](https://zookeeper.apache.org/doc/r3.1.2/zookeeperStarted.html#sc_RunningReplicatedZooKeeper)（必需）
  * 正在运行的 Hadoop 集群（推荐）
  * 使用 DNS（推荐）
